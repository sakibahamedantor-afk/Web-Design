# Web-Design Project

This is a monorepo for the e-commerce MVP.

## Structure

- `frontend/` — Customer-facing store (Next.js + Tailwind)
- `backend/` — REST API (Node.js + Express + PostgreSQL)

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```
Visit [http://localhost:3000](http://localhost:3000)

### Backend

```bash
cd backend
npm install
npm run dev
```
Backend runs on [http://localhost:4000](http://localhost:4000)

### Environment Variables

- Copy `.env.example` to `.env` in both folders and fill in values.

## Deployment

- Frontend: Vercel recommended
- Backend: Railway or Supabase for DB, Render for Node hosting
